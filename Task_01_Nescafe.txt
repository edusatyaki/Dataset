The Nescafe dataset contains sales, customer, product, and marketing information for a beverage business across different markets and regions. It includes details such as product types, sales, profit, expenses, inventory, and budget figures, along with customer location and order date. This dataset is useful for practicing data cleaning, data analysis, reporting, and business performance analysis using Excel, SQL, or Python.

| Column Name    | Description                                                     |
| -------------- | --------------------------------------------------------------- |
| Area Code      | Telephone area code representing the customer’s region          |
| Date           | Numeric date value representing the transaction time period     |
| Market         | The business market region where the sale occurred              |
| Market Size    | Classification of the market (e.g., Major Market, Small Market) |
| Product        | Name of the product sold                                        |
| Product Line   | Category of product (e.g., Beans, Leaves)                       |
| Product Type   | Type of product such as Coffee, Tea, or Espresso                |
| State          | State where the sale was made                                   |
| Type           | Whether the product is Regular or Decaf                         |
| Budget COGS    | Planned (budgeted) cost of goods sold                           |
| Budget Margin  | Planned (budgeted) profit margin                                |
| Budget Profit  | Planned (budgeted) profit amount                                |
| Budget Sales   | Planned (budgeted) sales amount                                 |
| COGS           | Actual cost of goods sold                                       |
| Inventory      | Quantity of product available in stock                          |
| Margin         | Actual profit margin earned                                     |
| Marketing      | Marketing expense spent on the product                          |
| Profit         | Actual profit earned from the sale                              |
| Sales          | Actual sales revenue                                            |
| Total Expenses | Total cost including COGS and marketing                         |
| Order Date     | Date when the customer placed the order                         |
| Customer Name  | Name of the customer who purchased the product                  |
| City           | City where the customer is located                              |
| Promo Code     | Promotional code used in the purchase (if any)                  |
| Full Product   | Combined or complete product description name                   |
